1 case_example
这个目录下是官方测试用例，你可以使用这些用例调测。
用法:
c/c++  ./cdn         case_example/case0.txt  your_path/result.txt
java   ./startup.sh  case_example/case0.txt  your_path/result.txt  
详情请参考SDK-gcc.zip或SDK-java.zip中的readme.txt

2 SDK-gcc.zip
c/c++ 开发框架包，你需要基于此包完成代码编写，详情参考文件中的readme.txt

3 SDK-java.zip
java 开发框架包，你需要基于此包完成代码编写，详情参考文件中的readme.txt
